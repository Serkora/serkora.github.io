!skip (t) — Skip the song (after time t). More

!skip (t) — Skip the song (after time t). Less

Bouncers and above can use this command on any track, ordinary users can use it on their own tracks.

If t >= 15, then t is assumed to be seconds, otherwise — minutes.
If t <= 0 or no t provided, then skips instantly.
If t == stop, then cancels the current skip timer.
If t == error, then skips the song and moves the dj to the head of the queue.
If t is not a number (except for the above), nothing happens.

 !skip 3.25 ==> skips the song in 3 minutes and 15 seconds.
!skip 125 ==> skips the song in 2 minutes and 5 seconds.
!skip ==> skips the song right away.

!add command response — Add a custom bot repsonse. More

!add command response — Add a custom bot repsonse. Less

Any user can add a command, unless it already exists.

command cannot contains spaces.
command must be unique.
response cannot start with ! or /, except for /me.
response may contain links and/or emoji

 !add hello Greetings, my friends! ; !hello ==> Greetings, my friends!
